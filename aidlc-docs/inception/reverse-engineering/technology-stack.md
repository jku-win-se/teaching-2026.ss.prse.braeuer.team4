# Technology Stack

## Programming Languages
- **Java** - 21 - Primary application and test implementation language
- **FXML** - JavaFX 21 compatible - Declarative UI layout for the smart-home and calculator views
- **CSS** - JavaFX stylesheet syntax - Visual styling for the desktop UI
- **XML** - Log4j configuration and Maven POM metadata
- **YAML** - GitHub Actions workflow automation
- **Markdown** - Project documentation and AI-DLC artifacts

## Frameworks and Libraries
- **JavaFX** - 21 - Desktop UI framework and FXML runtime
- **Log4j 2** - 2.23.1 - Logging framework configuration
- **JUnit** - 4.13.2 - Unit testing for calculator logic

## Build Tools
- **Maven** - Project build, dependency management, packaging, testing, and plugin execution
- **JaCoCo Maven Plugin** - 0.8.12 - Coverage report generation
- **Maven PMD Plugin** - 3.22.0 - Static analysis and duplication checks
- **GitHub Actions** - Managed workflow service - CI automation for compile, test, PMD, packaging, and report publication

## Runtime and Delivery
- **JDK** - 21 - Compile and runtime target
- **JavaFX Maven Plugin** - 0.0.8 - Simplifies JavaFX launch and packaging

## Infrastructure
- **Local Desktop Runtime** - Primary deployment model
- **Mock MQTT Configuration** - Demonstration-only integration settings stored in memory
